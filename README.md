# DSA Assignment 2

Solutions for COMP202 – Data Structures & Algorithms Assignment #2.

## Programs

Stacks & Expressions: Balanced parentheses, infix → postfix evaluation

Linked Lists: Reverse traversal, doubly linked list (insert/delete)

Graphs: BFS, DFS, Dijkstra’s algorithm

Heaps & Sorting: Min/max heap, random number sorting (Bubble, Selection, Insertion, Merge)

## Files

prog_<n>.c – Source code

doc_<n>.md – Program documentation

## How to Run

Compile and run each program:

gcc prog_1.c -o prog_1
./prog_1
